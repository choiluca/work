transform

  1. 이동옵션 : translate(x축, y축); , translateX(단위), translateY(단위)
  2. 회전옵션 : rotate(Ndeg); , rotateX(20deg), rotateY(30deg), rotateZ(30deg)
  3. 크기옵션 : scale(x축, y축), scaleX(), scaleY() (단위 : 배)
  4. 모서리각도조정 : skew(Xdeg, Ydeg), skewX(), skewY()
  5. 행렬 : matrix(scaleX(), skewY(), skewX(), scaleY, translateX(), translateY())

transition

  • transition-duration : 변환되는 속도를 지정
  • transition-delay : 변환대기 시간
  • transition-property : width height
  • trasition-timing-function :
    • linear : 처음부터 끝까지 같은 속도 (0, 0, 1, 1)
    • ease : 처음, 끝을 부드럽게 (기본값) (0.25, 0.1, 0.25, 1)
    • ease-in : 처음을 느리게
    • ease-out : 마지막을 느리게
    • ease-in-out : 처음, 마지막을 느리게
    • cubic-bezier(n, n, n, n) : 속도를 직접 셋팅
  • transition: property duration timing-function delay
